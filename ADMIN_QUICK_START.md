# Admin Panel Quick Start Guide

## Opening Admin Settings

1. Look for the **Settings button** (gear icon) in the **bottom-right corner**
2. Click it to open the Admin Settings Panel
3. Panel appears as a modal overlay

---

## What You Can Do (No Code Required!)

### 1. Change Profile Image
```
Admin Panel → Profile Image → Upload New Image
```
- Click upload area or drag & drop
- Select JPG, PNG, or WebP (max 5MB)
- Image updates immediately
- Can delete and upload new one anytime

### 2. Toggle Dark/Light Mode
```
Admin Panel → Theme → Click Moon/Sun Icon
```
- Light Mode: White background, emerald accents
- Dark Mode: Dark background, gold accents
- Theme remembered when you return
- Entire website updates instantly

### 3. Update Contact Information
```
Admin Panel → Contact Information → Edit Fields
```
- Edit: Name, Headline, Email, Phone, LinkedIn, GitHub
- All changes appear across the site
- Click "Save Contact Information"
- Changes persist in your browser

### 4. Upload Your Resume
```
Admin Panel → Resume Management → Upload Resume
```
- Click upload area or drag & drop
- Select PDF, DOC, or DOCX (max 10MB)
- File metadata shows automatically
- Can view, download, or delete

### 5. Show/Hide Resume Section
```
Admin Panel → Resume Section → Click Show/Hide
```
- Toggle visibility of entire resume section
- Navigation updates automatically
- Setting persists in browser
- No page refresh needed

## Troubleshooting

### Image upload not working?
- Check file size (max 5MB)
- Try JPG format
- Clear browser cache
- Try different image

### Contact info not saving?
- Click the Save button
- Check email format
- Check URL format (https://...)
- Try simpler text

### Resume not uploading?
- Check file size (max 10MB)
- Use PDF, DOC, or DOCX
- Try smaller file
- Clear browser cache

### Theme not changing?
- Try refreshing page
- Check browser dark mode setting
- Try different browser
- Clear browser cache

### Forgotten contact details?
- Open admin panel
- All stored data is there
- Can update anytime
- Changes apply immediately

## What's Saved vs What's Not

### Permanently Saved (localStorage)
- Theme preference
- Profile image (as base64)

### Not Saved (Session Only)
- Uploaded resume file (cleared on page reload)

### To Keep Permanently
- Save your resume file locally
- Note your contact information
- Backup your profile image
