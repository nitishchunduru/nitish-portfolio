HOW TO USE - QUICK START
================================================================================

OPENING ADMIN PANEL
1. Look for Settings button (gear icon) in BOTTOM-RIGHT corner
2. Click it
3. Admin settings modal opens

UPLOADING PROFILE IMAGE
1. Admin Panel → Profile Image
2. Click upload area or drag/drop
3. Select JPG/PNG/WebP (max 5MB)
4. Image appears immediately in hero section

TOGGLING DARK MODE
1. Admin Panel → Theme
2. Click moon or sun icon
3. Entire site switches themes
4. Preference remembered

UPDATING CONTACT INFO
1. Admin Panel → Contact Information
2. Edit name, email, phone, LinkedIn, GitHub
3. Click "Save Contact Information"
4. All changes appear site-wide instantly

UPLOADING RESUME
1. Admin Panel → Resume Management
2. Click upload area
3. Select PDF/DOC/DOCX (max 10MB)
4. Metadata auto-displays
5. Download/view available

HIDING RESUME SECTION
1. Admin Panel → Resume Section
2. Click Show/Hide button
3. Section toggles immediately
4. Navigation updates automatically

CLOSING ADMIN PANEL
1. Click "Close Admin Panel" button
2. Or click outside the modal
3. Or press ESC key
