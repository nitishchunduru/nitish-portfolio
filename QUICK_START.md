## 5-Minute Setup

### 1. Upload Your Profile Image (2 minutes)

```
1. Click Settings button (⚙️ gear icon) in top right
2. Click "Upload Image" button
3. Select your professional headshot (JPG, PNG, or WebP)
4. Drag sliders to position your face:
   - Zoom: Make yourself bigger/smaller
   - Vertical: Move up/down
   - Horizontal: Move left/right
5. Watch preview in circle to see result
6. Click "Save Image" when happy
```

**Pro Tip:** Center your face and make sure it fills the circular area for best appearance.

---

### 2. Update Your Contact Info (2 minutes)

```
1. Open Settings (⚙️ gear icon)
2. Find "Contact Information" section
3. Update:
   - Name
   - Email
   - Phone
   - LinkedIn URL
   - GitHub URL
4. Click "Save Contact Information"
5. Done! All changes appear site-wide instantly
```

---

### 3. Toggle Dark Mode (30 seconds)

```
1. Click moon 🌙 or sun ☀️ icon in top navigation
   (Or open Settings and click theme button)
2. Entire site switches instantly
3. Your preference is saved automatically
```

## Admin Features (Optional)

To access upload/replace/delete features:

```
1. Open Settings (⚙️ gear icon)
2. Click "Admin Mode: OFF" to turn it ON
3. Now you see:
   - Upload/Replace/Delete buttons for resume
   - Advanced image editor
   - Contact information editor
```

**Note:** Visitors don't see these controls. Only you see admin features.

## FAQ

**Q: Can visitors see my admin controls?**
A: No. Only you see upload/delete buttons. Visitors see final published version.

**Q: Where is my data stored?**
A: In your browser's localStorage. It's private and never leaves your computer.

**Q: Do my changes apply immediately?**
A: Yes. Everything updates instantly across entire portfolio.

**Q: Can I undo changes?**
A: Yes. Just upload a new image or edit the field again with correct information.

**Q: What if image won't upload?**
A: Check file size (max 5MB), format (JPG/PNG/WebP), or try different browser.

**Q: Can I use dark mode for printing?**
A: Light mode works better for printing. Switch to light before printing.

**Q: How do I delete everything?**
A: Click "Delete Image" button. To reset contact info, edit fields back to original values.
